compileBabel(argopt) → {Object}

Transpile JavaScript files using Babel with babel-loader.

Supports the following LJAS presets:

  • "default": Sets up @babel/preset-env with the modules option set to false
  • "react": Sets up @babel/preset-env (with modules set to false) & @babel/preset-react. If the BABEL_ENV environment variable is set to "development", then @babel/preset-react's development option is enabled.
  • "ts": Sets up @babel/preset-env (with modules set to false) & @babel/preset-typescript. Because Babel does not support type checking, Fork TS Checker Webpack Plugin is enabled.
